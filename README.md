<p>Installation:</p>
<p><code>npm install</code></p>
<p>To Run Test Suite:</p>
<p><code>npm test</code></p>
<p>To Start Server:</p>
<p><code>npm start</code></p>

<p>`<a href="https://emoviehub.netlify.app/">https://emoviehub.netlify.app/</a></p>
